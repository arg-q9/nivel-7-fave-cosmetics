#  Fave Cosmetics - E-commerce de Alto Impacto

Este proyecto es una recreación profesional de un sitio web de cosméticos, enfocada en la organización de código escalable, el uso de gráficos vectoriales (SVG) y una experiencia de usuario fluida con JavaScript Vanilla.

## 🚀 Características Técnicas
- **Estructura Multipage:** Organización profesional mediante carpetas (`/pages`, `/css`, `/js`, `/animacion`).
- **Hero Slider:** Implementación de un carrusel dinámico con lógica de `translateX` para un rendimiento óptimo.
- **Interactividad Avanzada:** Sistema de "Favoritos" (Wishlist) con manejo de estados mediante clases de CSS y eventos de JavaScript.
- **Gráficos Optimizados:** Uso de SVGs inline para garantizar nitidez en cualquier resolución y reducir los tiempos de carga.
- **Diseño Premium:** Estética oscura con detalles en `palevioletred` y efectos de profundidad mediante `inset box-shadow`.

##  Organización del Proyecto
- `index.html`: Punto de entrada y sección principal.
- `/pages/`: Vistas de Carrito, Favoritos y Gestión de Cuenta.
- `/animacion/`: Scripts específicos para componentes visuales como el Hero.
- `/css/`: Estilos globales y específicos del sitio.

##  Tecnologías Utilizadas
- HTML5 Semántico
- CSS3 (Flexbox, Grid, Custom Properties)
- JavaScript ES6+ (Manipulación del DOM y Timers)
